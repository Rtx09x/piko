# Changelog

## 0.1.0 - 2026-05-12

Initial Piko MVP.

- Added Chrome Manifest V3 extension shell.
- Added popup app for goal setting, recent context, sleep controls, and chat.
- Added Gemini REST integration.
- Added local browser activity tracking.
- Added variable nudge engine.
- Added optional recent browser-history import with explicit permission.
- Added privacy settings for allowed domains, blocked domains, and URL detail.
- Added content-page Piko overlay.
- Added extension icons.
- Added privacy policy draft and Chrome Web Store packaging notes.
